# 📊 Social Media Ad Analytics Dashboard

## Project Overview
Analysis of 1,143 Facebook ad records to identify high-performing 
audience segments and campaign efficiency using Python, SQL and Power BI.

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (Pandas, NumPy) | Data cleaning and feature engineering |
| SQL (SQLite) | Business queries and aggregations |
| Power BI | Interactive dashboard |

## Key Metrics Engineered
| Metric | Formula | Meaning |
|--------|---------|---------|
| CTR | Clicks / Impressions | Ad attractiveness |
| ROAS | Conversions / Spent | Return on ad spend |
| CPA | Spent / Approved Conversions | Cost per customer |
| Conversion Rate | Approved Conversions / Clicks | Sales efficiency |

## Key Findings
- 30-34 age group has the highest ROAS (0.34) — most efficient audience
- 45-49 age group has the highest spend ($20,750) but lowest ROAS (0.14)
